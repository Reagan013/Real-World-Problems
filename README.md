# REAL WORLD PROBLEM TutorMind 🧠

TutorMind is an adaptive AI learning tutor interface designed to run locally in the browser. It features a responsive, clean design that dynamically updates content based on user interactions.

## 🚀 Features
**Adaptive AI Learning:** Runs a local simulation of a tutoring agent.
**Session Controls:** Dynamic fields to customize name, subject matter, and starting skill levels.
**Gamified Elements:** Live score tracking and streak monitoring built directly into the UI.
**Modern Themes:** Outfitted with custom CSS variables tailored for clean layouts.

## 🛠️ Tech Stack
**HTML5:** For the semantic structure of the tutoring dashboard.
**CSS3:** Built with custom variables (:root), full Flexbox responsiveness, and precise element layouts.
